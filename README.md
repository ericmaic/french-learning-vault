# 一. 快捷键列表：

- Command + Enter：在连接处直接打开链接到网页或者到tab页
- Command + L：从原始的文字，依次切换到todo，todo done
- Command + 点击编辑书页图标：切换编辑和只读模式在新tab中建立
- Command + P：打开命令面板
- Command + Shift + D：新建日记
- Command + D：删除光标行
- Command + N：新建一个卡片，或者笔记
- Command + O：搜索文件的名字
- Command + ，：打开设置面板
- Command + Left：隐藏/展开左侧栏
- Command + Right：隐藏/展开右侧栏
- Command + E：预览模式与编辑模式切换
- Command + F：查找功能
- Command + 鼠标滚轮：字体大小（要提前设置）
- Command + "+"/"-"：同理
- "---,---": 包裹的是文件的属性
- / + 其他: 可以调出命令

# 二. 标签用法

- 一级标签：#标签名
- 二级标签：#标签名/二级标签（例：#提示词/源码）
- 文件分类做总体归纳，标签做细致归纳

## 1. 标签名称汇总：

- #语法
- #词汇
- #发音 

# 三. Markdown语法

- “# 标题”：一级标签
- “## 标题”：二级标签
- “### 标题”：三级标签
- “#### 标题”：四级标签
- “##### 标题”：五级标签
- “###### 标题”：六级标签
- “**内容**”
